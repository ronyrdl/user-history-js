# Weeks - JavaScript

Este repositorio contiene diferentes ejercicios y prácticas realizadas por semanas utilizando JavaScript, HTML y LocalStorage.

## 📂 Estructura del proyecto

### Semana 1

* Uso de `prompt`, validaciones y estructuras condicionales.
* Solicita nombre y edad al usuario.
* Verifica si el usuario es mayor o menor de edad.

### Semana 2

* Manejo de objetos y arreglos en JavaScript.
* Validación de productos.
* Uso de ciclos como `forEach` y `for in`.
* Gestión básica de datos.

### Semana 3

* Aplicación de notas usando:

  * DOM
  * Eventos
  * LocalStorage
* Permite agregar y guardar notas en el navegador.

### Semana 4

* CRUD de productos.
* Uso de:

  * API REST (`json-server`)
  * LocalStorage
  * Manipulación del DOM
* Incluye archivos como:

  * `db.json`
  * `script.js`
  * `index.html`

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* LocalStorage
* JSON Server

## ▶️ Cómo ejecutar

1. Clonar o descargar el proyecto.
2. Abrir cada carpeta semanal.
3. Ejecutar los archivos `index.html` en el navegador.

Para la semana 4 ejecutar también:

```bash
json-server --watch db.json --port 3000
```

## ✨ Objetivo

Practicar conceptos fundamentales de JavaScript como:

* Variables
* Condicionales
* Funciones
* Arreglos y objetos
* Eventos
* DOM
* LocalStorage
* Consumo de APIs
